Primeiros passos com HTML - Estrutura basica do HTML

voltar

A Estrutura basica do HTML é formada por 3 blocos principais onde um dos blocos ("html") engloba os outros dois ("head & body") e indica para o browser que este e um documento HTML.

Um documento HTML é formado por tags que possum nome especifico e estao posicionadas entre os sinais < e > . Algumas tags apresentam sinal de abertura e de fechamento enquanto outras apresentam apenas sinal de abertura.

A função das tags é informar ao navegador a estrutura do site atraves de comando indicados pelo nome daquela tag especifica.

As tags tambem podem conter atributos em seu interior que tem por finalidade informar e definir caracteristicas adicionais a uma determinada tag.

A seguir serão definidas algumas das tags que compoem a estrutura de um HTML bem como sua finalidade na pagina.

TAGS H1 a H6

voltar

Estas são as tags de cabeçalho. Normalmente sao utilizados para identificar paginas e seções e possuem aparencia diferenciada do restante do texto.

Existem seis niveis de cabeçalhos que podem ser usadas no HTML, que sao utilizados atraves das tags h1 a h6 sendo a tag h1 a mais relevante e tag h6 a menos relevante.

Recomenda-se que uma pagina tenha apenas uma tag h1 que indentifique o assunto da pagina, visto que os motores de busca web utiliza essa tag como palavra chave principal durante uma busca.

TAG P

voltar

Parágrafos de texto são gerados na HTML por meio das tags "p".

Esse é um exemplo de tag cuja disposição na tela se dá em forma de bloco, ou seja, um parágrafo é posto sempre abaixo do outro.

TAGS DE FORMATAÇÃO DE TEXTO (b, i, strong, sub, sup, mark, small)

voltar

As tags de formatação de texto ajudam a destacar trechos da parte escrita da página.

As tags b e strong sao usadas para colocar o texto em negrito/texto forte. apesar de visualmente identicas apresentam significados diferentes.

A tag b deve ser usado quando queremos destacar um texto apenas visualmente. enquanto a tag strong é utilizado para dar destaque visual e semantico ao texto demonstrandua sua relevancia.

A tag i é utilizada para dar enfase ao texto e coloca-lo em italico.

A tag mark tem a função de fazer uma marcação ao texto escolhido.

As tags sup e sub são utilizadas para efeitos de sobrescrito e subscrito respectivamente.

TAGS DE LISTA (ol, ul, li)

voltar

Listas são elementos úteis para organizar e ordenar itens que estão relacionados de alguma forma.

No HTML é possível criar três tipos de listas: ordenadas (com a tag ol), não ordenadas (com a tag ul), e de definição (por meio da tag dl).

A tag li se refere a cada item que faz parte de uma lista ordenada (tag ol) e nao ordenadas (tag ul).

TAG A

voltar

Links são normalmente utilizados para direcionar o usuário para outras páginas, ou para outras partes da mesma página.

Nos dois casos, utilizamos a tag a, que possui o atributo href no qual indicamos o destino daquele link.

Para adicionar um link para um elemento na mesma página é necessario adicionar o atributo id ao elemento igual àquele indicado no href.

É preciso lembrar nesse caso o href requer ainda o sinal de cerquilha (#) antes do id do elemento que será o foco do link.

TAG HR

voltar

O elemento HTML hr representa uma quebra temática entre elementos de nível de parágrafo.

Nas versões anteriores do HTML, representava uma linha horizontal.

TAG BLOCKQUOTE

voltar

O Elemento HTML blockquote (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação.
